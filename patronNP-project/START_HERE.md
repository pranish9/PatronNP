## рҹ”җ Authentication Flows

### Sign In/Sign Up
```
1. User enters email + password
2. Validation checks
3. Mock or API call
4. Store token in localStorage
5. Redirect to home
```

### Password Recovery
```
1. User clicks "Forgot Password"
2. Enters email
3. Gets redirected to OTP page
4. Enters 6-digit OTP
5. Gets redirected to password reset
6. Enters new password
7. Success & redirects to sign in
```
