PatronNP Frontend

PatronNP Frontend is a modern, responsive, and mobile-first web application built for the PatronNP creator monetization platform. The frontend provides an interactive user experience for creators, supporters, and administrators, enabling features such as creator subscriptions, donations, digital product purchases, live-stream alerts, analytics dashboards, and community engagement.

Built using React.js and Tailwind CSS, the application focuses on clean UI/UX, fast performance, scalability, and seamless integration with local Nepali payment systems like eSewa, Khalti, and IME Pay.

Features
Creator profile and dashboard
Supporter donation & subscription system
Digital product marketplace UI
OBS-compatible live stream alert interface
Real-time notifications
Analytics and revenue dashboard
Mobile-first responsive design
Secure authentication and role-based access
Tech Stack
React.js
Tailwind CSS
Axios
React Router
REST API Integration
Goal

To create a localized creator-support ecosystem for Nepal that enables creators to monetize content and engage with supporters effectively.
